This page embeds a Google Map demo from https://google-developers.appspot.com/ (e.g. cross-domain) using the following:

<iframe src="https://google-developers.appspot.com/..."
        allow="geolocation https://google-developers.appspot.com"></iframe>

The iframe's demo attempts to use the Geolocation API to display the user's current location. However, when the parent page uses a more restrictive feature policy (FeaturePolicy: geolocation 'self'), the iframe is blocked from accessing geolocation.
